# 开发流程

## 准备工作

1. 安装 koa 脚手架

```bash
npm install -g koa-generator
```

2. 安装 cnpm

```bash
npm install -g cnpm
```

3. 初始化项目

```bash
koa2 -e single_sign_on
cnpm install
```

4. 添加 .gitignore 和 .npmignore 配置文件

```text
.DS_Store
.vscode
node_modules/
```

5. 添加 README.md 说明文件

6. 添加 LICENSE 版权文件
